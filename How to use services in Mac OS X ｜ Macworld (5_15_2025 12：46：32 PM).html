A: Services can be installed in three places. First, they can be inside applications. Second, applications can install them in the /Nameofstartupdisk/Library/Services folder. Finally, some applications install them in the Library/Services folder in your user folder. Some of these may be .service files, but others may be automator actions, with a .workflow extension.

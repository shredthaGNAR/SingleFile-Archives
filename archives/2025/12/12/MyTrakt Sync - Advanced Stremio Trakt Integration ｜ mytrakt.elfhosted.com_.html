Feature Overview

Discover what's available now and what's coming next in your Trakt Stremio experience.

Current FeaturesAvailable Now

Powerful features already available in your Trakt Stremio addon:

Trakt.tv OAuth Integration
Sign-in with Trakt accounts and access personal data securely
Personal catalogs support
Watchlist, Watched, Collected, Favorites, Recommendations and other personal catalogs
Continue Watching & Up-Next support
Continue watching lists and optional UpNext filtering for unwatched episodes
Mark-as-watched stream
Optional stream to mark items as watched in Trakt after playback
Watch status display
Show checked/watched status (works when using TMDB as meta provider)
TMDB enrichment with key rotation
Batch fetch TMDB metadata with API key rotation and language fallback
IMDb / Cinemeta fallback
Use Cinemeta as a fallback for IMDB metadata provider
RPDB integration
User-specific posters, poster styles, rating overlays, and per-user poster customization
Fanart.tv integration
High-quality backgrounds, logos, and additional artwork from Fanart.tv
Top posters selection
Generate stunning movie posters with multiple rating badges from IMDb, Letterboxd, Rotten Tomatoes, and more.
MDBList integration
Import personal MDBList lists and add MDBList lists by URL; caching and management included
AniList (anime) integration
Search and import AniList lists and metadata where available
Subreddit lists support
Add subreddit-based catalogs (Reddit discovery integrations)
Multiple search providers
TMDB, OMDB, TVDB, Trakt, TVmaze, MAL, AniList and more for flexible searching
Robust caching & performance
Database-backed caching for catalogs, metadata and provider caches with configurable TTLs
Client & server pagination
Support for API pagination where possible and client-side pagination with stable sorts
Flexible sorting options
Rank, Last Watched, Added, Title, Year, Rating, Released/Aired, Runtime, Popularity, Random, Last Aired (TV)
Configurable catalog extras
Control per-catalog visibility: show or hide catalogs in Stremio Home, or make them available only in Stremio Discover
Custom lists & list management
Create custom catalog types, edit list names, reorder lists, control visibility and prefixing
TMDB localized / language selection
Metadata localized with English fallbacks and image language selection logic
Security & settings UI
Passkey authentication, setup dashboard, and fine-grained per-user settings
Ready to Use

All features are fully operational and ready for you to enjoy. Simply configure your addon and start using them right away!

Import shortcuts and Automator workflows on Mac

In the Shortcuts app for Mac, you can import shortcuts that were shared with you or convert an Automator workflow to a shortcut. Almost any existing Automator workflow can be migrated to a shortcut.

Note: If an Automator workflow is not supported in the Shortcuts app, an alert message appears telling you which action is unsupported.

Import a shortcut

To import an existing shortcut into the Shortcuts app on Mac, do one of the following:

  • Double-click a .shortcut file in the Finder.

  • Drag a .shortcut file into the main window of the Shortcuts app.

  • Drag a .shortcut file onto the Shortcuts app in the Dock.

Import an Automator workflow

To import an existing Automator workflow into the Shortcuts app on Mac, do one of the following:

  • Double-click a .workflow file in the Finder.

  • Drag a .workflow file into the main window of the Shortcuts app.

  • Drag a .workflow file onto the Shortcuts app in the Dock.

  • Control-click a .workflow file and choose Open With > Shortcuts.

The Automator workflow is converted to a shortcut and added to your collection in the Shortcuts app.

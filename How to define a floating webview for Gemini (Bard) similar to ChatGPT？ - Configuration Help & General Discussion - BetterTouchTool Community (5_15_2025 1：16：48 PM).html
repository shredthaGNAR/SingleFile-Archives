I have been using the ChatGPT floating webview since it was introduced in this blog post, and find it extremely useful.

I have been able to use the same preset and modify it to work with Bing Chat (bing.com/chat), but when I want to do the same with Gemini (Bard), I only see a black screen with a menu bar at the top. I tried bard.google.com in the past and gemini.google.com recently, but none of them works.

Here is a screenshot of how the Gemini floating webview looks:

Gemini2066×1464 88.3 KB

Using Gemini with the same account on Firefox works as expected.

Has anyone faced a similar issue and knows how to solve it?

Solved by Andreas_Hegenberg in post #2
in the advanced webview config, try to set the user agent to this: Mozilla/5.0 (Macintosh; Intel Mac OS X 14_3_1) AppleWebKit/605.1.15 (KHTML, like Gecko) Version/17.2 Safari/605.1.15 [image]

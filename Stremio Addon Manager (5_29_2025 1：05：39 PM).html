Effortlessly manage the order of your Stremio addons without re-installing.

Including the default Cinemeta catalogs.

Why?

Stremio currently doesn't allow users to change the order that their installed addons appear on the home screen. As a work around, it is common for users to remove and re-install addons in the order they want them to appear. This is a tedious and cumbersome process. This addon uses the internal Stremio API to manipulate addon order without having to add/remove them.

This is a workaround and not a solution. It is not recommended to use this tool unless you are comfortable with the risks involved. No support or warranty is given.

How?

  1. Connect your Stremio account. We support 2 different authentication methods:
    • Login using your Stremio account (Facebook login is not supported)
    • Login using an authentication key
      • Login to https://web.stremio.com/ using your Stremio credentials in your browser.
      • Open the developer console (?) and paste the follow code snippet: JSON.parse(localStorage.getItem("profile")).auth.key
      • Take the output value and paste it into the form below.
  2. Click 'Load Addons' to load your profiles addons.
  3. Re-order your addons as you like.
  4. Click the 'Sync To Stremio' button to sync the changes back to your profile.

Frequently Asked Questions

What risks are involved?

This was whipped together in a couple of hours. It is not well tested and may break your Stremio profile. Stremio functionality relies on the existence of certain addons (ie, Cinemeta). It is possible that the order of addons is also important to some functionality.

There is currently no way to 'reset' your addons to previous configurations or default configurations.

Are my credentials safe?

Yes. We only use your credentials to authenticate with the Stremio API. We do not store them in any way; as soon as you refresh the page they are gone.

If you're still in doubt, we highly encourage you to go read the source code. You'll see there's no hidden funny business going on.

What is the developer console and how do I open it?

The browser developer console is a tool which logs the information associated with a web application, such as network requests and errors. It also allows you to interact with the loaded web page using JavaScript. Most modern browsers have a developer console built in.

Use this guide to open your browsers developer console, ensuring have the Stremio web app open.

Can I use this to move Cinemeta catalogs from the top?

Yes! This will allow you to move the Cinemeta provided catalogs (ie, Popular - Series/Movie and Featured - Series/Movie) from the top of your catalog list.

Do note that the in-built 'Continue watching' catalog cannot be moved from the top.
